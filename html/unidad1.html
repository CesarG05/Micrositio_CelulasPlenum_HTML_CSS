Tema 1.1: Visión general del proceso de Ingeniería de Software

El proceso de Ingeniería de Software es un conjunto de actividades sistemáticas utilizadas para diseñar, desarrollar, probar, implementar y mantener software de alta calidad. Su objetivo es garantizar que el software sea confiable, eficiente y cumpla con los requisitos del cliente.
