El Protocolo de Transferencia de Hipertexto (HTTP) es el estándar de comunicación en la web. Define cómo los clientes (navegadores) y servidores intercambian información mediante solicitudes y respuestas.

Tema 3.2: HTML

El Lenguaje de Marcado de Hipertexto (HTML) es la base de las páginas web. Define la estructura del contenido mediante etiquetas como y permite la integración con CSS y JavaScript.
